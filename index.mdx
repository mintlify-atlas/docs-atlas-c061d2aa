Tienda Mi Cholo S.A.C is a full-stack point-of-sale and inventory management web application built on ASP.NET Core MVC (.NET 10). It provides everything a retail store needs to run daily operations: a real-time sales dashboard, receipt generation (Boleta and Factura), product and stock management, client tracking, and role-based access control for Administrators, Managers, and Cashiers.

## How it Works

<Steps>
  <Step title="Set up the database">
    Configure your SQL Server connection string in `appsettings.json` and run `dotnet ef database update` to apply migrations and seed initial data.
  </Step>
  <Step title="Log in">
    Open the app — it redirects to `/Acceso/Login`. Use the seeded administrator account (`admin@micholo.com` / `admin123`) to log in for the first time.
  </Step>
  <Step title="Add products and categories">
    Create product categories, then add your product catalog with purchase price, sale price, and initial stock levels.
  </Step>
  <Step title="Register sales">
    Use the New Sale screen to search products, build a cart, select a receipt type, and complete the transaction. Stock updates automatically.
  </Step>
</Steps>

<Note>
  The default administrator credentials seeded by the database (`admin@micholo.com` / `admin123`) should be changed immediately after the first login in any production deployment.
</Note>
